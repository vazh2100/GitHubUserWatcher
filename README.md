# GitHubUserWatcher

Тестовое задание от работодателя. Отображение информации о пользователях с сайта github.com

Паттерн: MVVM

Использованные библиотеки: RxJava2, Retrofit2, GSON, Room, LiveData, ViewModel, Navigation, Picasso, Material Design
